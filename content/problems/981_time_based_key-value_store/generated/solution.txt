## Solution

### Explanation

The problem asks us to find the number of columns that are not sorted lexicographically.

**Approach:**
1. Iterate through each column index
2. For each column, check if the characters from top to bottom are in non-decreasing order
3. If any column is not sorted, increment the counter
4. Return the final count

**Key Insight:** A column is sorted if for every consecutive pair of rows, the character in the upper row is less than or equal to the character in the lower row.

```python
from typing import List

class Solution:
    def minDeletionSize(self, strs: List[str]) -> int:
        # Number of rows and columns
        num_rows = len(strs)
        num_cols = len(strs[0])
        
        # Counter for columns to delete
        delete_count = 0
        
        # Iterate through each column
        for col in range(num_cols):
            # Check if column is sorted (non-decreasing order)
            for row in range(1, num_rows):
                # If current character is less than previous character
                # the column is not sorted
                if strs[row][col] < strs[row - 1][col]:
                    delete_count += 1
                    break  # No need to check further for this column
        
        return delete_count


# Test cases

```

### Complexity Analysis

| Complexity | Value | Description |
|------------|-------|-------------|
| **Time**   | O(n × m) | We iterate through each column (m) and for each column check all rows (n) |
| **Space**  | O(1) | We only use a counter variable, no extra data structures |

Where:
- `n` = number of strings (rows)
- `m` = length of each string (columns)

### Alternative Pythonic Solution

```python
from typing import List

class Solution:
    def minDeletionSize(self, strs: List[str]) -> int:
        # Using zip to iterate through columns and all() for checking order
        return sum(
            1 for col in zip(*strs)  # Transpose: each 'col' is a tuple of chars
            if col != tuple(sorted(col))  # Check if column is not sorted
        )
```

This one-liner uses `zip(*strs)` to transpose the grid, then checks if each column (now a row after transposing) is already sorted. If not, it counts it as a deletion.
